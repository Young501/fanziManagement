1 customers（客户基础信息表）

客户的基础档案信息，每个客户一条记录。

id
客户唯一ID（UUID主键）

company_name
客户公司名称

website_member_name
网站会员名 / 云系统账号名（例如云账房账号）

contact_person
客户联系人姓名

contact_info
联系人电话或联系方式

address
公司注册地址或办公地址

customer_status
客户状态
常见值：

active（在服务）

paused（暂停）

churned（已流失）

source_info
客户来源信息
例如：

老客户介绍

抖音

网站

线下

service_manager
负责该客户的财务或客户经理

created_at
客户创建时间

2 customer_company_profiles（企业工商信息）

客户公司的工商档案信息，一般一家公司只有一条。

id
记录ID

customer_id
客户ID（外键 → customers.id）

company_type
企业性质（有限公司 / 个体户 / 合伙企业等）

tax_rate
税率

invoice_agent
是否代开发票

community
所属小区 / 商圈

registration_date
公司注册日期

registered_capital
注册资本

tax_number
纳税人识别号

tax_officer
专管员姓名

tax_officer_phone
专管员电话

tax_office
所属税务所

legal_person
法人

supervisor
监事

finance_person
财务负责人

tax_handler
办税员

annual_revenue_last_year
去年营业收入

created_at
记录创建时间

updated_at
记录更新时间

3 customer_shareholders（股东信息）

记录客户公司股东结构，一家公司可多个股东。

id
记录ID

customer_id
客户ID（外键）

shareholder_name
股东姓名或公司名称

shareholder_type
股东类型
例如：

natural_person

company

share_ratio
持股比例（%）

subscribed_capital
认缴出资额

paid_in_capital
实缴出资额

share_order
股东顺序（股东1、股东2）

is_current
是否当前有效股东

note
备注

created_at
创建时间

updated_at
更新时间
